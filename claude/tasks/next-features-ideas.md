# 今後実装予定の機能メモ

このドキュメントは、次回以降の開発で実装を検討している機能のメモです。
優先度や詳細仕様は今後調整します。

## カレンダー機能

### 要件
- トレーニング記録をカレンダー形式で表示
- 各日付にトレーニングの概要を表示（何の種目をしたか）
- 日付をタップすると詳細ビューを表示
  - その日に行った全種目
  - 各種目のセット詳細（重量、回数）
  - インターバル情報
  - メモがあれば表示

### 技術的検討
- カレンダーライブラリ検討
  - `react-calendar` または `date-fns` + 自作
  - モバイルファーストのタッチ操作に対応
- 月表示・週表示の切り替え
- 当日以外の日付での記録追加機能
- 記録がある日のマーキング（ドット表示など）

### UI/UX
- カレンダーグリッドに種目名を簡潔に表示（例: ベンチ、スクワット）
- タップでモーダルまたはスライドアップで詳細表示
- 詳細ビューから編集・削除も可能に

## トレーニング記録フォームの改善

### セット削除機能
- 現状：セットの追加のみ可能
- 要望：セットの削除も可能にする
- 実装案：
  - 各セット行に削除ボタン（×アイコン）を追加
  - 最後の1セットは削除不可にする（最低1セットは必須）
  - 削除時に確認ダイアログは不要（誤操作防止よりもスピード重視）
