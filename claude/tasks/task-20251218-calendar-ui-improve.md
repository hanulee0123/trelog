# カレンダーUIの視認性改善

## 背景
カレンダーセル内の種目名表示において、長い種目名が省略（...）されることで視認性が低下している。
ユーザーより「省略のせいで種目が分かりづらくなるのが嫌」とのフィードバックあり。

## 要件
- 長い種目名でも可能な限り全文を表示する。
- セル内で改行する、またはフォントサイズを動的に調整するなどして視認性を確保する。
- 必要であればカレンダーセルの高さやレイアウト自体の見直しも検討する。

## 実装案
1. **複数行表示**: `white-space: normal` に変更し、セルの高さを拡張する。
2. **ツールチップ**: タップ/ホバーで全文を表示する（補助的手段）。
3. **リスト表示モード**: カレンダー内のドット表示をやめ、その日のリストを下部に大きく表示するスタイルへの変更検討（抜本的変更）。

## メモ
現状は `flex` と `text-overflow: ellipsis` で制御しているが、これを改善する。
