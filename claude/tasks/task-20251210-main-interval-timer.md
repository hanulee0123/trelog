## 技術的な詳細

### 実装する機能
1. **インターバルタイマーコンポーネント** (`src/features/timer/IntervalTimer.tsx`)
   - カウントダウン機能（秒単位）
   - 開始/一時停止/リセットボタン
   - タイマー設定（プリセット: 30秒、60秒、90秒、120秒 + カスタム入力）
   - 視覚的なプログレスバー（円形）
   - タイマー終了時の音声通知

2. **状態管理**
   - `useState`: タイマーの残り時間、実行状態（idle/running/paused）、設定時間
   - `useEffect`: カウントダウンのインターバル処理
   - `useRef`: setIntervalのIDを保持してクリーンアップ

3. **UI設計**
   - 大きな数字表示（分:秒形式、例: 1:30）
   - 円形プログレスバー（SVGまたはCSS conic-gradient）
   - モバイルフレンドリーな大きめボタン
   - プリセット時間の選択ボタン
   - カスタム時間入力フィールド

4. **統合**
   - App.tsxの「テンプレート / インターバル / フォームチェック」セクションを置き換え
   - TrainingLogFormのインターバル秒数と連携可能にする（将来的な拡張）

5. **スタイル**
   - タイマー専用CSS（円形プログレス、大きな数字表示）
   - ボタンのホバー/アクティブ状態
   - レスポンシブ対応（モバイル/デスクトップ）
